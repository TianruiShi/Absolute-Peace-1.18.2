# Absolute Peace 1.18.2

## 介绍

绝对和平，和平模式下也能获得不可掉落物品和无法合成物品。

## 扩展模块

扩展模块将突破绝对和平的限制，将会围绕有困难度的模式下做出修改。

未来可能添加的扩展模块：

- [x] 安全的矿物世界
- [ ] 绝对和平专用合成工作台、熔炉、烟熏炉
- [x] 绝对和平专用方块、箱子、发光方块
- [x] 更加简单的物品合成配方、合成数量
- [x] 生物物品掉落盒
- [x] 蜘蛛恐惧症模式
